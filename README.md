# 智能生产计划排程与需求预测补货系统

## 项目简介

智能生产计划排程与需求预测补货系统是一款面向制造业的生产管理工具，基于灵活作业车间调度问题（FJSP）算法，实现自动化的生产排程和物料需求预测，帮助企业优化生产效率、确保交期准时、降低库存成本。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **状态管理**: Zustand
- **UI 组件**: Tailwind CSS
- **图表可视化**: 原生 SVG + CSS Grid/Flexbox
- **数据导出**: SheetJS (xlsx)
- **构建工具**: Vite
- **日期处理**: Day.js

## 功能介绍

### 1. 生产排程
- 智能排程算法：基于遗传算法和优先级规则的多目标优化
- 甘特图可视化：直观展示生产计划时间线
- 产线负载分析：实时监控各产线利用率
- 支持批量订单自动排程

### 2. 订单管理
- 订单录入与编辑
- 优先级设置（1-5级）
- 交期管理
- 订单状态跟踪（待排程/已排程/已完成）

### 3. 产线管理
- 多产线配置与管理
- 产线产能设置
- 工序能力定义
- 产线状态监控

### 4. 产品与工序
- 产品工艺路线定义
- 工序标准工时管理
- 工序能力需求配置

### 5. 物料需求预测
- 安全库存设置
- 补货区域管理
- 库存状态预警（充足/偏低/紧急）
- 需求计算与补货建议

### 6. 系统设置
- 工作时间配置
- 排程权重调整
- 预测天数设置

## 使用说明

### 1. 初始配置

首次使用时，系统会加载默认测试数据，包括：
- 5条产线（冲压线A/B、焊接线、喷涂线、组装线）
- 3种产品及其工艺路线
- 8个待排程订单
- 6种物料及其库存状态

### 2. 日常操作流程

1. **管理订单**：在订单管理页面添加、编辑或删除订单
2. **配置产线**：根据实际产线情况配置产能和工序能力
3. **执行排程**：点击"执行排程"按钮，系统自动生成生产计划
4. **查看甘特图**：在排程结果页面查看和调整生产计划
5. **物料补货**：根据物料预警状态进行补货操作

### 3. 排程算法说明

系统采用基于优先级规则的启发式算法，综合考虑：
- 订单优先级（数值越小优先级越高）
- 订单交期紧迫度
- 产线负载均衡
- 工序能力匹配

## 部署方法

### 环境要求

- Node.js 18+
- npm 9+ 或 yarn 1.22+

### 安装步骤

```bash
# 克隆项目后进入目录
cd 规划与排程

# 安装依赖
npm install

# 开发模式启动
npm run dev

# 生产环境构建
npm run build

# 预览生产构建
npm run preview
```

### 构建产物

构建完成后，dist 目录包含可直接部署的静态文件，可部署到任意 Web 服务器或 CDN。
